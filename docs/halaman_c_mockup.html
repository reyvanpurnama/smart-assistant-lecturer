Teks Soal Uraian:

"Tuliskan sintaks query SQL untuk menampilkan nama mahasiswa (student_name), nama mata kuliah (course_name), dan nilai akhir (grade) yang diambil dari tabel mahasiswa, matakuliah, dan KRS. Kuncinya adalah hanya menampilkan data mahasiswa yang memiliki nilai di atas 80 dan menggunakan klausa JOIN secara benar. Jelaskan alur eksekusi query tersebut."

Konten Teks Jawaban:

Berikut adalah jawaban saya untuk tugas Praktikum 2:

1. Query SQL:

SELECT m.student_name, mk.course_name, k.grade
FROM mahasiswa m
INNER JOIN krs k ON m.id = k.mahasiswa_id
INNER JOIN matakuliah mk ON k.matakuliah_id = mk.id
WHERE k.grade > 80;

2. Penjelasan Alur Eksekusi Query:

  • Pertama, DBMS akan memproses klausa FROM untuk menentukan tabel utama yang digunakan, yaitu tabel mahasiswa (dengan alias m).
  • Kedua, DBMS melakukan INNER JOIN pertama ke tabel krs (alias k) berdasarkan kecocokan primary key m.id dan foreign key k.mahasiswa_id. Hasil join ini menghasilkan relasi mahasiswa beserta data mata kuliah yang diambilnya.
  • Ketiga, dilakukan INNER JOIN kedua ke tabel matakuliah (alias mk) menggunakan kolom k.matakuliah_id dan mk.id untuk mengambil informasi nama mata kuliah yang bersesuaian.
  • Keempat, baris data hasil gabungan tersebut akan difilter menggunakan klausa WHERE dengan kondisi k.grade > 80. Sehingga data nilai yang kurang dari atau sama dengan 80 akan dibuang.
  • Terakhir, klausa SELECT dieksekusi untuk memproyeksikan atau menampilkan tiga atribut akhir saja, yaitu nama mahasiswa, nama mata kuliah, dan nilai akhir.
